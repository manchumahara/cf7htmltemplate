Upload `cf7htmltemplate` to the `/wp-content/plugins/` directory2. Activate the plugin through the 'Plugins' menu in WordPress3. This plugin need contact form 7 installed to work== Frequently Asked Questions ==== Screenshots ==1. Template Configuration2.
